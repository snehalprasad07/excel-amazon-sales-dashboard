# excel-amazon-sales-dashboard
This project presents an Amazon Sales Analysis Dashboard built in Excel to track and evaluate sales performance, order trends, customer demand, and operational efficiency across India. The dashboard integrates raw transactional data and transforms it into interactive visual insights such as monthly sales trends, top-performing products, city-wise orders, fulfilment performance, and size demand distribution. It enables stakeholders to monitor KPIs like revenue, order volume, delivery performance, cancellations, and returns to support data-driven decision-making.
